## 📖 项目简介

**校内勤工俭学智能匹配平台**是一款专为高校学生设计的兼职信息发布与申请小程序。平台连接校内岗位发布方（老师、实验室、校内商户）与求职学生，提供从岗位发布、智能推荐、在线申请、录用处理到签到评价的全流程闭环服务。

与普通兼职平台不同，本项目突出 **“智能匹配”** 与 **“校园可信”** 两大特色，通过学工认证构建信任基础，利用 LBS 距离计算和信誉评分优化推荐结果，让兼职更安全、更高效。

> 🎓 本项目遵循完整软件开发流程，文档规范，代码开源。

## ✨ 功能特性

| 模块 | 功能点 | 说明 |
|------|--------|------|
| **用户系统** | 微信一键登录 | 快速注册/登录，无需繁琐注册 |
| | 学生/发布方双角色 | 学生申请兼职，发布方发布岗位 |
| | 学号/工号认证 | 校内身份认证，确保真实可信 |
| **岗位管理** | 岗位发布 | 标题、类型、报酬、时间、地点、描述 |
| | 分类浏览 | 按工作类型筛选（快递、家教、实验室等） |
| | 关键词搜索 | 快速查找目标岗位 |
| **智能匹配** | LBS距离计算 | 基于工作地点与学生位置推荐最近岗位 |
| | 信誉分排序 | 根据历史评价推荐优质发布方/学生 |
| | 个性化推送 | 订阅感兴趣的岗位类型 |
| **申请流程** | 在线申请 | 一键提交申请，实时反馈 |
| | 发布方审核 | 查看申请列表，录用/拒绝 |
| | 状态跟踪 | 待处理、已录用、已完成全程可查 |
| **签到打卡** | 扫码签到 | 发布方提供二维码，学生扫码确认到岗 |
| | GPS定位打卡 | 可选，确保工作真实完成 |
| **评价体系** | 双向互评 | 完成后互相打分评论 |
| | 信誉分动态更新 | 影响后续匹配优先级 |
| **消息中心** | 微信订阅消息 | 申请结果、录用通知、打卡提醒 |

## 🚀 快速开始

### 环境要求
- 已注册[微信小程序](https://mp.weixin.qq.com/)并获取 AppID
- 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 安装 [Git](https://git-scm.com/)（可选，用于版本管理）

### 本地运行步骤

1. **克隆项目到本地**
   ```bash
   git clone https://github.com/shuichangdonghly/campus-work-study.git
   ```
   或直接下载 ZIP 包解压。

2. **导入微信开发者工具**
   - 打开微信开发者工具，点击“导入项目”
   - 选择项目目录，填入你的 AppID
   - 点击“确认”

3. **开通云开发环境**
   - 在开发者工具顶部点击“云开发”，开通云环境
   - 记录环境 ID

4. **配置云开发环境 ID**
   - 打开 `miniprogram/app.js`，找到 `wx.cloud.init`，将 `env` 替换为你的环境 ID
   - 示例：
     ```js
     wx.cloud.init({
       env: '你的环境id',
       traceUser: true
     });
     ```

5. **创建云数据库集合**
   - 在云开发控制台 -> 数据库，创建以下集合：
     - `users`（用户表）
     - `jobs`（工作表）
     - `applications`（申请表）
   - 可根据需要设置索引（如 `jobs` 的 `status` 字段）

6. **上传并部署云函数**
   - 右键点击 `cloudfunctions` 下的每个云函数文件夹，选择“上传并部署：云端安装依赖”
   - 确保所有云函数都成功部署

7. **编译运行**
   - 点击开发者工具的“编译”按钮，即可在模拟器中预览
   - 可使用手机扫码体验真机效果

## 📱 使用说明

### 学生角色
1. **注册/登录**：首次使用点击“微信一键登录”，选择“学生”角色，填写学号、姓名等信息完成认证。
2. **浏览岗位**：首页展示所有招聘中的岗位，可按类型筛选或搜索关键词。
3. **申请岗位**：进入岗位详情页，点击“立即申请”，等待发布方审核。
4. **查看申请状态**：在“我的申请”中可看到所有申请记录及当前状态（待处理、已录用、已完成）。
5. **签到打卡**：被录用后，工作开始时需扫描发布方提供的二维码或进行 GPS 打卡。
6. **评价**：工作完成后，可在订单中对发布方进行评价。

### 发布方角色
1. **注册/登录**：选择“发布方”角色，填写工号（或营业执照）、姓名等信息认证。
2. **发布岗位**：填写岗位信息（标题、类型、报酬、时间、地点、描述）后提交。
3. **管理申请**：在“我发布的”中点击具体岗位，可查看所有申请者，并进行“录用”或“拒绝”操作。
4. **确认完成**：学生工作结束后，点击“确认完成”，订单状态变更为已完成。
5. **评价学生**：完成后可对学生进行评价。
